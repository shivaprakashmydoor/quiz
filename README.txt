Solution to find the longest compound word using python.
To run: python lcw.py <input file>

Solution: Read the words from the input file and form a Trie tree, and also add the words to a hast table. \
Next step is to get the word from the hash table and iterate through the tree and record the longest
compound word. Update the longest compound word as we iterate word by word. Finally what we have is the
longest compound word
